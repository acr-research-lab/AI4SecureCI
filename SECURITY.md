
# Security Policy

This repository must not contain malware binaries, live samples, credentials, cloud secrets, private datasets, or unsafe operational instructions.

## Allowed Content

- Sanitized logs
- Derived feature examples
- Educational explanations
- High-level lab descriptions
- Safe notebooks using benign or sanitized data
- Figures, diagrams, and reports

## Prohibited Content

- Malware samples
- Weaponized scripts
- Exploit code intended for misuse
- Cloud credentials
- Private keys
- Raw student data
- Raw sensitive telemetry
- VM images containing malware
