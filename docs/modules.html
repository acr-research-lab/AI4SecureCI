Modules

The AI4SecureCI - AI for Secure Cloud Infrastructure curriculum is organized into eight connected modules. Module 0 builds the foundation in cloud cyberinfrastructure, AI/ML, and fair AI. Modules 1 through 7 then move students through cloud network security, automated access control, malware detection and classification, adversarial robustness, fairness-aware security, and explainable AI.

Module summary

This table summarizes the materials, videos, and student-facing output for each module.

For questions about the instructor materials, please contact the Associated Faculty members.

Module no. Module title Materials Video Student output
0 Getting Started with Cloud CyberInfrastructure, Machine Learning, and Fair AI Download (ZIP) Coming soon The students will configure a cloud environment, build basic machine learning models, and perform an initial bias and fairness analysis using fairness tools.
1 Cloud CI Network Security Download (ZIP) Slide Lab The students will train and evaluate a cloud network intrusion detection system using captured network traffic.
2 Authorization and Automated Access Control in Cloud CI Download (ZIP) Coming soon The students will develop a deep-learning-based access control model that makes authorization decisions using user and resource metadata.
3 Online Malware Detection in Cloud CI Download (ZIP) Slide Lab The students will build a defensive machine learning detector that classifies cloud workload behavior as benign or malicious.
4 Classifying Malware Threats in Cloud CI Download (ZIP) Slide Lab The students will train a malware family classifier using behavioral API-call data to identify malware types or families.
5 Adversarial Attacks and Defenses in Cloud CI Download (ZIP) Lab The students will generate adversarial examples against malware detection models and retrain defensive models to improve robustness.
6 Bias and Fairness in Cloud CI Security Download (ZIP) Coming soon The students will audit a cloud-security model for bias and apply fairness-aware mitigation methods to improve decision quality.
7 Explainable AI for Cloud CI Security Download (ZIP) Slide Lab The students will use SHAP-based local and global explanations to interpret the decisions made by cloud-security machine learning models.

These modules were developed through a joint effort provided by faculty and students at Tennessee Technological University in Cookeville, TN, North Carolina Agricultural and Technical State University in Greensboro, NC, and SUNY at Albany, funded by grants provided by the U.S. National Science Foundation.
